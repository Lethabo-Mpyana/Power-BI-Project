# Power BI Dashboard: Data Professional Survey

## 📊 Project Overview
This project follows Alex the Analyst’s Power BI tutorial series.  
The dashboard analyzes survey responses from data professionals, covering job titles, programming languages, salaries, and work-life balance.

## 🔄 Workflow Summary
### Data Preparation
- Imported raw Excel dataset into Power BI.
- Cleaned columns (`Job Title`, `Programming Language`) using delimiters.
- Standardized numeric data by creating an **Average Salary** column.
- Converted data types for accurate calculations.

### Dashboard Construction
- KPI Cards: Total survey takers, average age.
- Stacked Bar Chart: Average salary by job title.
- Clustered Column Chart: Popular programming languages.
- Tree Map: Geographic breakdown of participants.
- Gauge Charts: Sentiment scores (work-life balance, salary happiness).
- Donut Chart: Salary averages across genders.

### Refinement & Styling
- Applied custom theme and layout for professional presentation.
- Renamed headers and labels for clarity.

## 📈 Key Insights
- Data Analysts report lower average salaries compared to Data Scientists.
- Python and SQL dominate as the most popular programming languages.
- Work-life balance sentiment is higher than salary satisfaction.

## 🚀 How to Use
1. Download the `.pbix` file.
2. Open in [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
3. Explore the interactive visuals.
